# Guacamole-on-WSL-2
Setup Guacamole on WSL 2

Update the WSL 2 Linux kernel (optional)

https://docs.microsoft.com/en-us/windows/wsl/wsl2-kernel

Install Ubuntu

## Update Ubuntu

`sudo su`

`apt-get update`

`sudo apt ugrade`

## Install Vnc Server

apt-get install vnc4server

## Install xfce

apt-get install xfce

## Setup Vnc password

Exit root 

`exit`

`vncpasswd`
